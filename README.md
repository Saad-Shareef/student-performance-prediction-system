# 🎓 Student Performance Prediction System

An AI-powered web application that predicts student academic performance using Machine Learning and provides personalized recommendations.

## Features

### Student Module
- Secure Registration & Login
- Performance Prediction
- Personalized Recommendations
- Prediction History
- Performance Analytics Dashboard

### Teacher Module
- View Student Predictions
- Performance Distribution Analytics
- Risk Analysis Dashboard

### Admin Module
- User Management
- Student Statistics
- System Analytics

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript
- Chart.js

### Backend
- Flask
- SQLAlchemy
- Flask-Login

### Machine Learning
- Scikit-Learn
- NumPy
- Joblib

### Database
- SQLite

## Machine Learning Features

- Study Hours
- Attendance
- Sleep Hours
- Mental Health
- Social Media Usage
- Internet Quality
- Diet Quality
- Exercise Frequency
- Extracurricular Activities

## Performance Classes

- Excellent
- Moderate
- Poor

## Installation

```bash
git clone https://github.com/Saad-Shareef/student-performance-prediction-system.git

cd student-performance-prediction-system

pip install -r requirements.txt

python app.py
```
